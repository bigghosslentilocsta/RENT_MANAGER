# Clear Cache Instructions

If you're still seeing old data after normal refresh, follow these steps:

## On Mobile (Android/iOS)

### Chrome (Android)
1. Open Chrome menu (3 dots)
2. Tap **Settings**
3. Tap **Privacy and security**
4. Tap **Clear browsing data**
5. Select **Cached images and files**
6. Tap **Clear data**
7. Reload the app

### Safari (iOS)
1. Go to **Settings** > **Safari**
2. Scroll down and tap **Clear History and Website Data**
3. Confirm by tapping **Clear History and Data**
4. Reopen the app

### Alternative (Any Mobile Browser)
1. Close the app/tab completely
2. Reopen your browser
3. Visit the app URL again
4. The app should load fresh data

## On Desktop/Laptop

### Easy Method (Works on All Browsers)
Press: **Ctrl + Shift + R** (Windows/Linux) or **Cmd + Shift + R** (Mac)

### Chrome/Edge
1. Press **Ctrl + Shift + Delete** (Windows) or **Cmd + Shift + Delete** (Mac)
2. Select **Cached images and files**
3. Click **Clear data**
4. Reload the page

### Firefox
1. Press **Ctrl + Shift + Delete** (Windows) or **Cmd + Shift + Delete** (Mac)
2. Select **Cache**
3. Click **Clear Now**
4. Reload the page

## Note
After the latest update, the app should automatically fetch fresh data every 10 seconds. If you still see stale data, it may be a browser caching issue that requires manual clearing as described above.

## For PWA (Installed App)
If you installed the app as a PWA:
1. Uninstall the PWA
2. Clear browser cache (see instructions above)
3. Reinstall the PWA from the browser
